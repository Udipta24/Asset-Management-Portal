## ✨ Features

* 🔐 Role-based access (User, Asset Manager, Maintenance Engineer)
* 📦 Asset management with category & subcategory
* 🔄 Request system:

  * New asset requests
  * Maintenance requests
* 📊 Status tracking (Pending, Approved, Rejected, Fulfilled)
* 💬 Remarks system for communication
* 🛠 Maintenance tracking (cost, type, next due date)
* ⚡ Dynamic asset assignment (fetching available assets)
* 🌙 Light & Dark mode UI

---
